🏥 Pulse-Path: Universal Emergency Navigator
Pulse-Path is a real-time, global hospital locator and medical facility finder. This application was developed to provide users with instant access to healthcare locations based on satellite data, ensuring that help is always reachable.
🚀 Live Demo
Check out the live app here: https://pulse-path-mmvjt5fcygag3ynowhcfnj.streamlit.app/
✨ Core Features
Global Geocoding: Converts any city, landmark, or street name into precise GPS coordinates using the Nominatim engine.
Satellite Mapping: Visualizes nearby medical facilities on an interactive dark-mode satellite map.
Smart Recommendations: Automatically identifies and highlights the closest medical facility with estimated arrival times.
Detailed Facility Info: Expandable list providing full addresses and specific location data for every result found.
🛠️ Technical Stack
Frontend: Streamlit (Python-based Web Framework)
Geospatial Data: Geopy & OpenStreetMap API
Data Handling: Pandas & Requests
Deployment: GitHub for Version Control & Streamlit Community Cloud for Hosting
